Steps to run the code: 
	Visual studio should be installed.
1> Git clone this repo:
	https://github.com/SreaSrestha9/CrossBrowserHackathon.git

2> Navigate to HackathonApplitools folder and double click on HackathonApplitools.sln. This will open the project in Visual Studio.

3> From Nuget Package install lastest stable version:
	> Eyes.Selenium 
	> Microsoft.NET.Test.Sdk
	> Nunit
	> Nunit3TestAdapter
	> Selenium.Firefox.Wendriver
	> Selenium.WebDriver
	> Selenium.WebDriver.ChromeDriver
	> Selenium.WebDriver.MicrosoftDriver
	> Selenium.Support

4> Build the solution.

5> To run the modern tests for version1:
	> Go to TestCases> ModernApproach>ModernTestsV1.cs > Run


6> To run the modern tests for version2:
	> Go to TestCases> ModernApproach>ModernTestsV2.cs > Run

7> To run the traditional tests for version1:
	> Go to TestCases> TraditionalApproach > HomeTraditoinalV1 > Run

8> To run the traditional tests for version2:
	> Go to TestCases> TraditionalApproach > HomeTraditoinalV2 > Run
